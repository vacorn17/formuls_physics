Формулы по физике


Скорость при равномерном движении :
• v = S / t

Средняя скорость :
• v_ср = (S₁ + S₂ + ...) / (t₁ + t₂ + ...)

Скорость при равноускоренном движении :
• v = v₀ + at

Путь при равноускоренном движении :
• S = v₀t + (at²)/2

Без времени (уравнение движения) :
• S = (v² - v₀²) / (2a)

Координата при равномерном движении :
• x = x₀ + vt
Координата при равноускоренном движении :
• x = x₀ + v₀t + (at²)/2

Угловая скорость :
• ω = φ / t = 2π / T = 2πν

Линейная скорость при вращении :
• v = ωR

Центростремительное ускорение :
• a_цс = v² / R = ω²R

Ускорение при криволинейном движении :
• a = √(a_τ² + a_цс²)

Второй закон Ньютона :
• F = ma

Сила тяжести :
• F_тяж = mg
Сила упругости (Закон Гука) :
• F_упр = k|Δx|

Сила трения скольжения :
• F_тр = μN

Сила трения покоя :
• 0 ≤ F_тр.покоя ≤ μN

Сила всемирного тяготения :
• F = G * (m₁m₂)/r²

Ускорение свободного падения на высоте :
• g_h = G * M / (R + h)²

Первая космическая скорость :
• v₁ = √(gR) = √(GM/R)

Вторая космическая скорость :
• v₂ = √(2) * v₁

Вес тела (неподвижное или равномерное) :
• P = mg

Вес тела (ускорение вверх) :
• P = m(g + a)

Вес тела (ускорение вниз) :
• P = m(g - a)

Невесомость :
• P = 0 при a = g

Кинетическая энергия :
• E_к = (mv²)/2

Потенциальная энергия в поле тяжести :
• E_п = mgh

Потенциальная энергия упругой деформации :
• E_п = (k(Δx)²)/2

Механическая энергия :
• E = E_к + E_п

Закон сохранения механической энергии :
• E_к1 + E_п1 = E_к2 + E_п2

Работа силы :
• A = F * S * cosα

Теорема о кинетической энергии :
• A = ΔE_к = (mv₂²)/2 - (mv₁²)/2

Мощность :
• P = A / t = F * v

Коэффициент полезного действия :
• η = (A_полезн / A_затр) * 100%

Импульс тела :
• p = mv

Импульс силы :
• FΔt = Δp

Закон сохранения импульса :
• m₁v₁ + m₂v₂ = m₁u₁ + m₂u₂

Упругий удар (1D) :
• v₁′ = ((m₁-m₂)v₁ + 2m₂v₂)/(m₁+m₂)

Неупругий удар :
• u = (m₁v₁ + m₂v₂)/(m₁+m₂)

Условие равновесия тел :
• ΣF = 0, ΣM = 0

Момент силы :
• M = F * l

Плечо силы :
• l — расстояние до оси вращения

Правило рычага :
• F₁ * l₁ = F₂ * l₂

Давление :
• p = F / S

Давление столба жидкости :
• p = ρgh

Сила Архимеда :
• F_A = ρgV_погр

Условие плавания тел :
• F_A = P или ρ_жидк > ρ_тела

Закон Паскаля :
• p = F₁/S₁ = F₂/S₂

Расход жидкости :
• Q = v * S = const

Количество вещества :
• ν = m / μ = N / N_A

Число Авогадро :
• N_A ≈ 6.022·10²³ моль⁻¹

Масса молекулы :
• m_0 = μ / N_A

Основное уравнение МКТ :
• p = (2/3) * n * E_ср

Средняя кинетическая энергия молекулы :
• E_ср = (3/2) * kT

Температура по шкале Кельвина :
• T(K) = t(°C) + 273.15

Постоянная Больцмана :
• k ≈ 1.38·10⁻²³ Дж/К

Средняя квадратичная скорость :
• v_кв = √(3kT/m₀) = √(3RT/μ)

Уравнение Менделеева-Клапейрона :
• pV = νRT

Универсальная газовая постоянная :
• R ≈ 8.31 Дж/(моль·К)

Изобарный процесс (Закон Гей-Люссака) :
• V/T = const

Изохорный процесс (Закон Шарля) :
• p/T = const

Изотермический процесс (Закон Бойля-Мариотта) :
• pV = const

Уравнение Клапейрона :
• (p₁V₁)/T₁ = (p₂V₂)/T₂

Первое начало термодинамики :
• Q = ΔU + A′

Работа газа :
• A′ = pΔV

Внутренняя энергия идеального газа :
• U = (i/2) * νRT

Теплоемкость при постоянном объеме :
• C_V = (i/2) * R

Теплоемкость при постоянном давлении :
• C_p = C_V + R

Адиабатный процесс :
• pV^γ = const, γ = C_p/C_V

КПД тепловой машины :
• η = (Q_нагр - Q_хол) / Q_нагр

КПД цикла Карно :
• η = (T_нагр - T_хол) / T_нагр

Закон Кулона :
• F = k * (|q₁q₂|)/r²

Коэффициент k :
• k = 1/(4πε₀ε)

Напряженность электрического поля :
• E = F/q

Напряженность точечного заряда :
• E = k * (|q|)/r²

Принцип суперпозиции полей :
• E = E₁ + E₂ + ...

Потенциал электрического поля :
• φ = E_п / q

Потенциал точечного заряда :
• φ = k * q / r

Напряжение (разность потенциалов) :
• U = A/q = φ₁ - φ₂
Связь U и E для однородного поля :
• U = E * d

Электроемкость проводника :
• C = q/φ

Электроемкость плоского конденсатора :
• C = (ε₀εS)/d

Энергия заряженного конденсатора :
• W = (qU)/2 = (CU²)/2 = q²/(2C)

Электрический дипольный момент :
• p = ql

Сила тока :
• I = q/t

Плотность тока :
• j = I/S

Закон Ома для участка цепи :
• I = U/R

Сопротивление проводника :
• R = ρ * l/S

Зависимость сопротивления от температуры :
• R = R₀(1 + αΔt)

Последовательное соединение :
• R_общ = R₁ + R₂ + ..., I = const

Параллельное соединение :
• 1/R_общ = 1/R₁ + 1/R₂ + ..., U = const

Закон Ома для полной цепи :
• I = ℰ/(R + r)

Работа тока :
• A = UIt = I²Rt = (U²/R)t

Мощность тока :
• P = UI = I²R = U²/R

Мощность источника :
• P_ист = Iℰ

Правила Кирхгофа (1-е) :
• ΣI_вх = ΣI_вых

Правила Кирхгофа (2-е) :
• Σℰ = ΣIR

Сила Ампера :
• F_A = I * B * l * sinα

Сила Лоренца :
• F_Л = q * v * B * sinα

Магнитная индукция :
• B = F_А/(I * l)

Магнитный поток :
• Φ = B * S * cosα

Индукция прямого тока :
• B = (μ₀μI)/(2πR)

Индукция в центре витка :
• B = (μ₀μI)/(2R)

Индукция соленоида :
• B = μ₀μ * n * I

Закон электромагнитной индукции (Фарадей) :
• ℰ_инд = -ΔΦ/Δt

ЭДС в движущемся проводнике :
• ℰ = B * v * l * sinα

Индуктивность :
• L = Φ/I

ЭДС самоиндукции :
• ℰ_си = -L * (ΔI/Δt)

Энергия магнитного поля :
• W = (L * I²)/2

Уравнение гармонических колебаний :

Скорость при гармонических колебаниях :

Ускорение при гармонических колебаниях :

Период колебаний математического маятника :

Скорость волны в струне :

Закон отражения :

Закон преломления (Снеллиуса) :

Абсолютный показатель преломления :

Условие минимума интерференции :

Уравнение Эйнштейна для фотоэффекта :
